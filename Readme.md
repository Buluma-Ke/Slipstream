# Slipstream 🏎️

F1 analytics web app powered by FastF1, Plotly Dash, and DuckDB.

> Work in progress.

---

## Setup

```bash
python -m venv .venv
.venv\Scripts\activate        # Windows
pip install -r requirements.txt
```
